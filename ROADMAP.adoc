= Social Media Tools Roadmap
:toc:
:toclevels: 3

== Overview

Social Media Tools is a monorepo containing analysis and verification tools for social media researchers and content moderators.

== Components

=== dipstick (40% complete)

*Status:* Active development +
*Purpose:* Social media content quality analysis tool

*Current features:*
- Basic content extraction
- Sentiment analysis pipeline
- Pattern detection algorithms

*Planned improvements:*
- Multi-platform API integrations
- Advanced NLP models
- Real-time monitoring

=== polygraph (20% complete)

*Status:* Early development +
*Purpose:* Fact-checking and verification framework

*Current features:*
- Verification rule DSL (in design)
- Basic claim extraction

*Planned improvements:*
- Integration with fact-checking databases
- Automated evidence gathering
- Credibility scoring system
